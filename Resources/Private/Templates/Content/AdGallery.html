This template is a gallery that requires jQuery + jQuery UI Easing and AD Gallery from here: http://adgallery.codeplex.com/documentation Download AD Gallery and include it like this with TypoScript: page { includeJS.adgallery = fileadmin/template/ADGallery/lib/jquery.ad-gallery.js includeCSS.adgalcss = fileadmin/template/ADGallery/lib/jquery.ad-gallery.css } Note: if you want to copy this HTML-file to your fileadmin-folder, you can specifty the folder like this: plugin.tx_camaliga.view.templateRootPath = fileadmin/template/files/ Copy this file in a subfolder named "Content" of this folder.
