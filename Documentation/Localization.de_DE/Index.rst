=========================================
EXT: Camaliga - carousel/map/list/gallery
=========================================

:Extension:
      camaliga

:Beschreibung:
      Camaliga carousel/map/list/gallery ist eine moderne und flexible Extension um Elemente auf verschiedene Art anzuzeigen. Es können Kategorien für Sortierung und Restriktionen genutzt werden. Außerdem kann man verschiedene jQuery-Plugins benutzen.
