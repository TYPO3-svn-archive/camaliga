This template is a jquery responsive Image Slider that requires jQuery and SKDslider from here: http://dandywebsolution.com/skdslider/ Download SKDslider and include it like this with TypoScript: page { includeJSFooter.skdslider = fileadmin/template/skdslider/src/skdslider.min.js includeCSS.skdslider = fileadmin/template/skdslider/src/skdslider.css } Start it with: page.jsFooterInline.795 = TEXT page.jsFooterInline.795.value = $('#demo1').skdslider({'delay':5000, 'animationSpeed': 2000}); Note: if you want to copy this HTML-file to your fileadmin-folder, you can specifty the folder like this: plugin.tx_camaliga.view.templateRootPath = fileadmin/template/files/ Copy this file in a subfolder named "Content" of this folder.
